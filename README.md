## Description

An interactive platformer simulation that implements background music (with optional mute button), and running/jumping sound effects.

---

## Setup and interaction

Run the program using Live Server

The user controls the fox by using arrow keys and WASD. The fox can jump onto the platforms and avoid the gaps in the ground to make it to the other side of the world.
